# A to Z Informatic Labs Docs         

This README.md contains instructions how to access this project, information about this project, and helpful links.     
<br>

## How to get started:       
Clone this project into a directory of your choosing using this code snippet.       
```git clone https://github.com/azinfolab/web.git```     
<br>

## Helpful Git tips:
* Remember to ```git pull``` before working on the project each time and before pushing new work to the reprository.
* To push new work to the repo use the following commands:        
      1. ```git add .```       
      2. ```git commit -m "Some meaningful message for the work you done"```        
      3. ``` git push```
